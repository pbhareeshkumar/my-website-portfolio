# Personal Portfolio Website

A portfolio website built using HTML, CSS, and JavaScript, 
created as part of IBM's "Introduction to HTML, CSS & JavaScript" course on Coursera.

# Built With
- HTML5
- CSS3
- JavaScript

# Features
- About Me section
- Skills showcase
- Projects section highlighting Data Analysis and Chatbot projects
- Interactive Recommendations section with a submit form
- Smooth navigation with anchor links and scroll-to-top button
